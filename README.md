# Braian Kleber | Engenharia de Software & Portfólio

Este é o repositório do meu portfólio pessoal e profissional estruturado para apresentar minhas habilidades técnicas, visão de arquitetura e projetos em andamento (como Klebsuchan, Kleber Finance e outras landing pages avançadas).

O projeto foi construído do zero, com foco extremo em **design**, **performance** e **animações fluidas**, estruturando uma fundação sólida de código combinada a um visual sofisticado e minimalista voltado para a conversão de clientes.

## 🚀 Tecnologias Utilizadas

A *stack* base deste projeto foi desenhada para promover rapidez de entrega com a máxima qualidade visual e tipagem forte:

- **[React 18](https://react.dev/)** - Biblioteca escalável de interface.
- **[Vite](https://vitejs.dev/)** - Build tool e ambiente de desenvolvimento ultrarrápido (HMR).
- **[TypeScript](https://www.typescriptlang.org/)** - Tipagem estática para robustez do código.
- **[Tailwind CSS](https://tailwindcss.com/)** - Framework utilitário, utilizado para criar o design system customizado diretamente no client.
- **[Motion (Framer Motion)](https://motion.dev/)** - Biblioteca poderosa de animações que dá vida a todas as transições de scroll e fadeIn do projeto.
- **[Lucide React](https://lucide.dev/)** - Biblioteca de ícones modernos e vetorizados.

## ⚙️ Arquitetura das Seções

- **Hero Section**: Introdução de impacto, alinhando engenharia e resolução de problemas estruturais.
- **Sobre & Educação**: Resumo focado na minha experiência (4+ anos de mercado), formação técnica (Gestão de TI concluída e ADS em andamento).
- **Stack Tecnológica**: Grid exibindo meu ecossistema principal de atuação (Tailwind, React, Next.js, Node.js, Prisma, PostgreSQL).
- **Workflow & Entregáveis**: Linha do tempo iterativa do processo (do escopo ao código e estabilização).
- **Showcase de Projetos**: Grade em *cards* apresentando a vitrine de produtos e infraestruturas, evidenciando as *stacks* utilizadas.
- **Social Proof**: Área reservada para feedbacks (Prova Social) e FAQ, engatilhado para remover o atrito e trazer clareza comercial ao cliente.

## 🛠️ Rodando Localmente

Para clonar e testar o ambiente em sua máquina local, execute:

1. **Clone o repositório**
   ```bash
   git clone https://github.com/braianklebercamargo-sketch/portfolio.git
   ```

2. **Acesse o diretório**
   ```bash
   cd portfolio
   ```

3. **Instale as dependências**
   ```bash
   npm install
   ```

4. **Inicie o ambiente de desenvolvimento**
   ```bash
   npm run dev
   ```

5. O Vite inicializará o servidor. Acesse `http://localhost:3000` / `http://localhost:5173` no navegador.

## 🚢 Hospedagem & Deploy (Vercel)

Esta aplicação foi polida e está **[Vercel Ready]**. A Vercel entende as configurações de build do ecossistema Vite nativamente.
* **Build Command**: `npm run build` (`vite build`)
* **Output Directory**: `dist`
No painel da Vercel, apenas importe sua conta no GitHub, selecione este repositório, e o deploy global de borda entra no ar automaticamente.
