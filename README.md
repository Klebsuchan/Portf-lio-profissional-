# Portfólio Profissional - Braian Kleber

![Portfólio Preview](/public/braianfoto.png) <!-- Aqui você pode adicionar um print do seu portfólio no futuro -->

Um portfólio moderno, rápido e responsivo construído com **React**, **TypeScript**, **Tailwind CSS** e **Vite**, focado em performance e UX/UI.

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias e ferramentas:

- **[React](https://reactjs.org/)**: Biblioteca JavaScript para construção de interfaces.
- **[TypeScript](https://www.typescriptlang.org/)**: Superset do JavaScript que adiciona tipagem estática.
- **[Vite](https://vitejs.dev/)**: Ferramenta de build extremamente rápida para projetos frontend.
- **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utility-first para estilização rápida e responsiva.
- **[Framer Motion](https://www.framer.com/motion/)**: Biblioteca para animações fluidas no React.
- **[Lucide React](https://lucide.dev/)**: Pacote de ícones minimalistas.

## 💻 Como rodar o projeto localmente

Para rodar este projeto em sua máquina, siga os passos abaixo:

### Pré-requisitos

Você precisará ter o [Node.js](https://nodejs.org/) e o [npm](https://www.npmjs.com/) (ou yarn, pnpm) instalados em sua máquina.

### Instalação

1. Clone este repositório:
   ```bash
   git clone https://github.com/Klebsuchan/SEU_REPOSITORIO.git
   ```

2. Acesse a pasta do projeto:
   ```bash
   cd SEU_REPOSITORIO
   ```

3. Instale as dependências:
   ```bash
   npm install
   ```
   *Ou utilizando yarn: `yarn install` / pnpm: `pnpm install`*

4. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

5. O servidor iniciará e você poderá acessar a aplicação pelo navegador no endereço `http://localhost:5173` (ou a porta informada no terminal).

## 🌍 Deploy

O deploy deste projeto está configurado para ser feito de forma automatizada pela [Vercel](https://vercel.com).
Os arquivos estáticos da pasta `public` (como a imagem `braianfoto.png`) são copiados automaticamente durante o processo de build (`npm run build`) e ficam disponíveis na raiz do projeto em produção.
